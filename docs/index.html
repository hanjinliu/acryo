acryo

acryo is an extensible cryo-EM/ET toolkit for Python.

Jump to GitHub

Installation

pip install acryo -U

Before getting start with acryo, it is highly recommended to understand the concept of out-of-core computation using dask, 3D rotation using scipy, and table data processing using polars.
